# Skills Reference

Detailed reference documentation for PAI skills. Each skill provides specialized capabilities that activate based on triggers in your requests.

## How to Use This Reference

Each skill page includes:
- **Triggers** — Phrases that activate the skill
- **Workflows** — Available procedures the skill can execute
- **Examples** — Sample usage and expected outputs
- **Integration** — How the skill works with other skills
